Bienvenido/a a la aplicacion para la gestion de clientes y productos entrantes y salientes del almacen. A continuacion,
le indicaremos como funciona esta aplicacion.

CONSIDERACIONES GENERALES

    Todos los numeros que deban introducirse en esta aplicacion son NUMEROS NATURALES (del 0 en adelante). Cualquier
    numero utilizado fuera de esta consideracion puede generar problemas en la aplicacion y/o forzar su cierre.

    Los menus se repetiran si los valores introducidos son diferentes a los mostrados en pantalla.

    Pese a que la aplicacion permite la introduccion de valores vacios, se recomienda encarecidamente eliminar aquellos
    clientes y/o productos con valores no validos.

    LOS DATOS DE LA APLICACION SE CARGAN AL INICIAR LA APLICACION Y SE GUARDAN AL CERRAR LA APLICACION EXITOSAMENTE.
    En caso de realizar cualquier accion que provoque el cierre inesperado de la aplicacion LOS DATOS NO SERAN GUARDADOS.

FUNCIONAMIENTO

    Consta de un menu principal donde podra elegir entre 6 opciones diferentes:

        1 - MOSTRAR ELEMENTOS: mostrara una lista de clientes y/o de productos segun se desee. Posee ademas una opcion
        que permite listar los productos segun sean entrantes o salientes.

        2 - AGREGAR ELEMENTOS: permite incluir nuevos clientes y/o productos segun se desee. Posee ademas una opcion
        para agregar productos segun sean entrantes o salientes sin necesidad de indicarlo previamente.

        3 - ELIMINAR ELEMENTOS: permite la eliminacion de clientes y/o productos DE FORMA INDIVIDUAL (a diferencia de la
        agregacion de elementos, que permite agregar mas de uno simultaneamente).
        A TENER EN CUENTA: eliminar un cliente ELIMINARA TODOS LOS PRODUCTOS QUE COINCIDAN CON EL IDENTIFICADOR DE
        DICHO CLIENTE.

        4 - ORDENAR PEDIDO: permite ordenar un pedido entrante o saliente segun los diferentes identificadores o los
        nombres de los productos.

        5 - MODIFICAR ELEMENTO: permite modificar un cliente o un producto segun sea entrante o saliente.
        NO SE PERMITE MODIFICAR LA ENTRADA O SALIDA DE UN PRODUCTO, solo se permite modificar los datos de dicho producto.

        6 - CERRAR APLICACION: cierra la aplicacion.
        RECUERDE, LOS DATOS SOLO SERAN GUARDADOS UNA VEZ LA APLICACION SE CIERRE DESDE ESTE MENU EXITOSAMENTE.
